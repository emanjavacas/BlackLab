Class ThreadPriority

  • java.lang.Object
    • nl.inl.util.ThreadPriority
  • All Implemented Interfaces:
    Prioritizable

    public class ThreadPriority
extends Object
implements Prioritizable
    Allows us to lower thread priority and/or pause thread. The thread must cooperate by calling behave() regularly. We don't use Java's own thread priority system because it's not particularly portable / practical to use (differences in priority between OS'es, needs root on Linux, etc.)
