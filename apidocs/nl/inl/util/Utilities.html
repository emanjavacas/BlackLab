      • createBlackLabTestDir

        public static File createBlackLabTestDir(String name)
        Create a temporary directory for BlackLab testing. A GUID is used to avoid collisions. Note that because of memory mapping and file locking issues, temp dirs may hang around. It is good practice to start and end a test run by calling removeBlackLabTestDirs().
        Parameters:
        name - descriptive name to be used in the temporary dir (useful while debugging)
        Returns:
        the newly created temp dir.
