Class Hit

  • java.lang.Object
    • nl.inl.blacklab.search.Hit
  • All Implemented Interfaces:
    Cloneable, Comparable<Hit>

    public class Hit
extends Object
implements Comparable<Hit>, Cloneable
    Class for a hit. Normally, hits are iterated over in a Lucene Spans object, but in some places, it makes sense to place hits in separate objects: when caching or sorting hits, or just for convenience in client code. This class has public members for the sake of efficiency; this makes a non-trivial difference when iterating over hundreds of thousands of hits.
