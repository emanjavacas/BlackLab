Class CompleteQuery

  • java.lang.Object
    • nl.inl.blacklab.search.CompleteQuery

  • public class CompleteQuery
extends Object
    Represents a combination of a contents query (a TextPattern) and a metadata "filter query" (a regular Lucene Query). This kind of query is produced by parsing SRU CQL, for example.
