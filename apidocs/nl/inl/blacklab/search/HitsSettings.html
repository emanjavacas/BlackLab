      • setConcordanceProperties

        public void setConcordanceProperties(String wordFI,
                            String punctFI,
                            Collection<String> attrFI)
        Indicate what properties to use to build concordances. This configuration is only valid when using forward indices to build concordances.
        Parameters:
        wordFI - FI to use as the text content of the <w/> tags (default "word"; null for no text content)
        punctFI - FI to use as the text content between <w/> tags (default "punct"; null for just a space)
        attrFI - FIs to use as the attributes of the <w/> tags (null for all other FIs)
