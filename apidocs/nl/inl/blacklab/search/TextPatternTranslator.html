Class TextPatternTranslator<T>

  • java.lang.Object
    • nl.inl.blacklab.search.TextPatternTranslator<T>
  • Type Parameters:
    T - the destination type
    Direct Known Subclasses:
    TextPatternTranslatorSpanQuery, TextPatternTranslatorString

    public abstract class TextPatternTranslator<T>
extends Object
    Interface for translating a TextPattern into a different representation. This uses the Visitor design pattern to recursively translate the whole TextPattern tree.
