        public int compare(Object a,
          Object b)
        Compares two hits on this property. The default implementation uses get() to compare the two hits. Subclasses may override this method to provide a more efficient implementation. Note that we use Object as the type instead of Hit to save on run-time type checking. We know (slash hope :-) that this method is only ever called to compare Hits.

        Returns:
        0 if equal, negative if a < b, positive if a > b.
