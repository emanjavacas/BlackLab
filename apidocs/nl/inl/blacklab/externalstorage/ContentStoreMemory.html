        public String[] retrieveParts(int id,
                     int[] start,
                     int[] end)
        Retrieve one or more substrings from the specified content. This is more efficient than retrieving the whole content, or retrieving parts in separate calls, because the file is only opened once and random access is used to read only the required parts. NOTE: if offset and length are both -1, retrieves the whole content. This is used by the retrieve(id) method.
